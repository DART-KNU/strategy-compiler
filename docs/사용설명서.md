# DART Strategy Compiler 사용설명서

경북대학교 금융데이터분석학회 DART 한국 주식 퀀트 전략 백테스트 엔진입니다.
자연어로 전략 아이디어를 말하면 AI가 구체화하고, 자동으로 백테스트를 실행합니다.

---

## 사전 준비

- Python 3.9 이상 설치 (Anaconda 권장)
- Git 설치
- **OpenAI API 키**

---

## 설치

### 1. 저장소 받기

```bash
git clone https://github.com/DART-KNU/strategy-compiler.git
cd strategy-compiler
```

### 2. DB 파일 다운로드

백테스트 데이터베이스(9.6 GB)를 아래 링크에서 받으세요.

👉 https://drive.google.com/file/d/1KY1NYZKzGbODoa8XxkKS7VCJ7Bf6RPz1/view?usp=sharing

다운받은 `backtest.db` 파일을 아래 경로에 넣어주세요.

```
strategy-compiler/
└── database/db/data/db/backtest.db
```

### 3. 가상환경 만들기

**Git Bash (Windows)**

```bash
python -m venv .venv
source .venv/Scripts/activate
pip install -r requirements.txt
```

**PowerShell (Windows)**

```powershell
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

**Mac / Linux**

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### 4. API 키 등록

`.env.example` 파일을 복사해서 `.env` 파일을 만들고, API 키를 입력합니다.

```bash
cp .env.example .env
```

`.env` 파일을 메모장으로 열어서 아래처럼 작성하세요.

```
OPENAI_API_KEY=sk-...본인키입력...
```

---

## 실행

가상환경이 활성화된 상태에서 아래 명령어를 실행합니다.

```bash
python -m backtest_engine.api.chat \
  --db database/db/data/db/backtest.db \
  --out runs/
```

실행하면 다음과 같은 화면이 나타납니다.

```
+------------------------------------------------------------+
|               DART Strategy Compiler Ver 1.0               |
|------------------------------------------------------------|
| 종료: Ctrl+C 또는 'exit'                                    |
+------------------------------------------------------------+

  안녕하세요, 경북대학교 금융데이터분석학회 DART ...

  어떤 전략을 만들어 볼까요?

>
```

## 사용 방법

`>` 프롬프트에 전략 아이디어를 자유롭게 입력하면 됩니다.
AI가 빠진 파라미터를 하나씩 물어보면서 전략을 완성합니다.

### 실제 대화 예시

```
> 모멘텀 전략 짜줘

  모멘텀 룩백 기간은 어떻게 설정할까요?
  (단기 1개월 / 중기 3~6개월 / 장기 12개월)

> 3개월

  몇 종목을 보유할 계획인가요?

> 10종목, 마르코위츠 비중 배분

  위험 회피 계수(gamma)는? 공격적이면 0.5, 기본 1.0, 보수적이면 3~5

> 기본으로

  리밸런싱 주기는? (주간 / 월간 / 분기)

> 주간

  ... (설정 완료 후 자동으로 백테스트 실행)
```

### 결과 출력 예시

모든 파라미터가 확정되면 백테스트가 자동 실행되고 결과가 출력됩니다.

```
+-- 백테스트 결과 --------------------------------------------------+
| [ 수익률 ]                                                       |
|   총 수익률 ............... +33.38%                              |
|   연환산(CAGR) ............ +10.44%                              |
|   벤치마크 수익률 ......... +142.99%                             |
| [ 위험 ]                                                        |
|   최대 낙폭(MDD) .......... -28.41%                             |
| [ 위험 조정 수익률 ]                                              |
|   샤프 비율 ............... 0.632                               |
+------------------------------------------------------------------+
```

결과 파일은 `runs/전략명__실행ID/` 폴더에 저장됩니다.

- `.html` — 차트 + 지표 + AI 리뷰가 담긴 리포트 (브라우저로 열기)
- `.json` — 전체 백테스트 데이터 (수익률 시계열 등)

## 지원 전략

| 전략 | 설명 |
|------|------|
| 모멘텀/반전 | 최근 수익률 기반, 1개월 스킵 옵션 |
| 멀티팩터 | 퀄리티 + 모멘텀 + 저변동성 등 복합 팩터 |
| 저변동성 | 역변동성 비중 배분 |
| 밸류/퀄리티 | 영업이익률, 매출성장, 순부채 등 재무 팩터 |
| 벤치마크 추종 | KOSPI200 추적오차 최소화 |
| 향상된 인덱스 | 벤치마크 추종 + 알파 팩터 혼합 |

## 자주 묻는 질문

**Q. 대화 중에 전략을 바꾸고 싶어요.**
A. 그냥 "다시 처음부터 해줘" 또는 새로운 전략 아이디어를 말하면 됩니다.

**Q. 결과 HTML 파일은 어떻게 열어요?**
A. `runs/` 폴더에서 `.html` 파일을 더블클릭하면 브라우저에서 열립니다.

**Q. API 키는 어디서 받나요?**
A. 운영진에게 문의하거나, https://platform.openai.com 에서 개인 계정으로 발급 가능합니다.

**Q. 오류가 나요.**
A. 대부분 DB 파일 경로 문제입니다. `database/db/data/db/backtest.db` 위치를 확인하세요.
